# Épreuves Combinées — Application PWA

## Fichiers inclus
- `index.html` — L'application principale
- `manifest.json` — Manifeste PWA (nom, icône, couleurs)
- `sw.js` — Service Worker (mode hors-ligne)
- `icon-192.png` / `icon-512.png` — Icônes de l'application

---

## Installation sur Android via GitHub Pages (gratuit, 5 min)

### Étape 1 — Créer un compte GitHub
→ https://github.com/signup (gratuit)

### Étape 2 — Créer un nouveau dépôt
1. Cliquez sur **"New repository"**
2. Nom : `epreuves-combinees` (ou autre)
3. Cochez **"Public"**
4. Cliquez **"Create repository"**

### Étape 3 — Déposer les fichiers
1. Cliquez **"uploading an existing file"**
2. Glissez-déposez les 5 fichiers : `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
3. Cliquez **"Commit changes"**

### Étape 4 — Activer GitHub Pages
1. Allez dans **Settings** → **Pages**
2. Source : **Deploy from a branch**
3. Branch : **main** / **root**
4. Cliquez **Save**
5. Attendez 1-2 minutes → votre URL sera : `https://VOTRE_PSEUDO.github.io/epreuves-combinees/`

### Étape 5 — Installer sur Android
1. Ouvrez l'URL dans **Chrome Android**
2. Une bannière **"Installer l'application"** apparaît automatiquement
3. Tapez **Installer** → l'icône apparaît sur votre écran d'accueil !

L'app fonctionne ensuite **hors-ligne** (pas besoin de connexion).

---

## Alternative : Netlify (encore plus simple)
1. Allez sur https://netlify.com → créez un compte gratuit
2. Glissez le **dossier entier** dans la zone de dépôt Netlify
3. Votre URL est générée instantanément
4. Ouvrez-la dans Chrome Android → installez !
